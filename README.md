# The Oldest Engine

A visual timeline tracing the history of the Three Life Drivers — Appetite, Approval, and Ambition — from ancient Egypt to the digital age.

By Steve Griffin · Compass Institute for Authentic Enterprise and Leadership

## Local Development

No build tools required. Open `index.html` directly in a browser:

```bash
open index.html
```

Or serve locally with any static server:

```bash
npx serve .
```

## Deploy to Netlify

1. Push this directory to a GitHub repository
2. Log in to [Netlify](https://netlify.com) and click **Add new site > Import an existing project**
3. Connect your GitHub repo
4. Netlify auto-detects the static site — no build command needed
5. Publish directory: `.` (root)
6. Click **Deploy site**

The included `netlify.toml` handles cache headers automatically.

## Custom Domain (optional)

In Netlify: **Site settings > Domain management > Add custom domain**

Suggested: `drivers.thecompassinstitute.com` or `timeline.aligniq.com`
